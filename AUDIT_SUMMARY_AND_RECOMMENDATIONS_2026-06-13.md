### Root Cause Analysis

**Primary cause (85% of failures):** Email addresses generated by `researcher.py` web scraping are **guessed** using patterns like:
- `training@bigcorp.com`
- `careers@company.ae`
- `formation@company.fr`

These do **not exist** at real companies. Bounce rates by type:

```
Generic training/ careers/ formation/ = 60–80% bounce
Oracle, IBM, Microsoft, SAP, Google = 100% bounce (portal only)
Small French/Réunion companies = 30–50% bounce (correct domain but wrong person)
MS365 groups = 100% reject (distribution list blocks external senders)
```

**Why this matters:** Every bounced email damages your domain reputation with Gmail/ProtonMail, reduces deliverability, and wastes effort.

**Solution implemented:** France Travail API provides **direct employer contact emails** with 0% guessing. Use it exclusively going forward.
